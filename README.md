# CivicAI
Open-Source AI for Democracy, Governance &amp; Public Intelligence

Empowering Governments, Citizens & Institutions with Local AI

CivicAI is an open-source AI platform built for democracy, governance, and civic systems in Nepal and developing countries worldwide.

We combine Small Language Models (SLMs), local civic datasets, and transparent AI systems to make governance:

More accessible
More transparent
More efficient
More inclusive
